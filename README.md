# SEMICONDUCTOR-IN-NANODEVICES
Exploring the role of semiconductor nanostructures (Si, GaAs, TMDCs) in electronics. Covers quantum confinement, tunable bandgaps, and applications from nanomedicine to neuromorphic computing.
